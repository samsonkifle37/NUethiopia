✨ Features Overview

Save & Organize Your Favorite Places

✨ Create Collections

Organize places into custom collections with emojis and colors. Perfect for organizing by type, trip, or theme.

❤️ Save Places

Save any place to your collections with one click. Add places to multiple collections instantly.

🎨 Customize

Add emoji icons and colors to your collections. Make your collections visually distinct and fun.

API Endpoints:

✅ All Implemented & Tested

Plan Multi-Day Trips with Activities

📋 Create Plans

Create multi-day itineraries (1-14 days). Automatically generates day placeholders with customizable titles.

📍 Add Activities

Add places as activities to specific days. Set time slots and add notes for each activity.

🔄 Organize

Drag-drop to reorder activities. Edit titles, descriptions, and publish status anytime.

API Endpoints:

GET /api/user/itineraries POST /api/user/itineraries GET /api/user/itineraries/[id] PATCH /api/user/itineraries/[id] DELETE /api/user/itineraries/[id] GET /api/user/itineraries/[id]/days POST /api/user/itineraries/[id]/days PATCH /api/user/itineraries/[id]/days/[dayId] DELETE /api/user/itineraries/[id]/days/[dayId] GET /api/user/itineraries/[id]/days/[dayId]/activities POST /api/user/itineraries/[id]/days/[dayId]/activities PATCH /api/user/itineraries/[id]/days/[dayId]/activities/[activityId] DELETE /api/user/itineraries/[id]/days/[dayId]/activities/[activityId] PATCH /api/user/itineraries/[id]/days/[dayId]/activities/reorder
✅ All Implemented & Tested

Share Itineraries with Friends & Family

🔗 Share Links

Generate unique shareable links. Set expiration dates (7 days, 30 days, or never).

📱 QR Codes

Generate scannable QR codes. Share via WhatsApp, email, or print them out.

📊 Analytics

Track view counts and who's interested. See when your itinerary was viewed.

🌐 Social Sharing

Share on Twitter, WhatsApp, and Facebook. Pre-filled messages with your itinerary link.

🔐 Privacy Control

Revoke links anytime. Choose who can see your itineraries.

👥 Public Viewing

Viewers don't need to log in. See beautiful timeline of activities.

API Endpoints:

GET /api/user/itineraries/[id]/share POST /api/user/itineraries/[id]/share DELETE /api/user/itineraries/[id]/share GET /api/itineraries/share/[token] (public - no auth needed)
